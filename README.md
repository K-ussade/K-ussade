## Release Status

Models, datasets, and Spaces referenced from this account are `planned` unless their Hugging Face repositories, cards, release notes, safety notes, and governance review status say otherwise.

| Status | Meaning |
| --- | --- |
| `planned` | Name, purpose, and review path are documented; no public artifact is claimed. |
| `scaffolded` | Public repo exists with README, card, or safety notes; no release claim unless stated. |
| `experimental` | Limited reviewed artifact exists for evaluation with explicit limitations. |
| `released` | Public artifact has approved card, safety notes, release notes, companion links, and review status. |
| `deprecated` | Artifact remains visible only with replacement, sunset, or archival context. |

## Public Boundary

Allowed public material includes public Foundation documents, public governance documents, public transparency reports, public program descriptions, public civic reference material, clearly labeled synthetic examples, approved sanitized public samples, open-source code specifically approved for public release, and model or dataset cards that describe limitations.

Forbidden public material includes donor private data, student data, school private data, volunteer private data, private Foundation operations, sealed YOSO-YAi LLC IP, customer data, unreleased model weights, private training corpora, secrets, tokens, API keys, precise sensitive infrastructure locations, and security-sensitive NEURONA operational details.

For AI operations and trading-agent governance surfaces, forbidden material also includes live autonomous agents, runnable orchestration, private prompts, private model outputs, trading bots, trading systems, trading strategies, signals, financial advice, investment advice, returns, balances, account data, performance claims, return claims, and client outcome claims.

## Release Discipline

Hugging Face is a public release surface for reviewed models, datasets, and Spaces. It is not the canonical home for sealed model development, private training corpora, unreleased model weights, private telemetry, or sensitive operational materials.

GitHub repositories linked from this profile should state owner, authority, status, canonical source posture, public/private boundary, required documentation, required graphs, refusal rules, and quality gates before they are treated as public project surfaces.
